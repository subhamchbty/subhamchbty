### 🧠 The short version

I build event-driven microservices and keep them reliable in production. Most of my time goes into the hard middle of a system: making jobs run **exactly once**, making payroll runs reconcile to the cent, making auth bulletproof, and shaving 90%+ off the queries everyone said were "just slow."

TypeScript ▸ Node.js ▸ NestJS  →  primary

Kafka ▸ RabbitMQ ▸ Redis        →  the nervous system
AWS EKS ▸ Docker ▸ K8s          →  where it all runs

### ⚡ Things I've shipped that I'm proud of

- 🔐 **Exactly-once at scale** — distributed locking with Redlock so queue jobs never double-fire across EKS pods
- 💸 **A payroll engine** — self-scheduling batch runs + a Redis-backed reconciliation pass that catches every mismatched cent
- 🧩 **A rulebook, untangled** — turned a monolithic nested MongoDB document into a parent/child rule tree, making search actually feasible
- 📈 **−97% query time** — when "it's just slow" turned out to be a missing index and a bad access pattern
